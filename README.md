# M11H-9900k-6800xt
Hackintosh on ROG maximus XI hero wifi + i9 9900k + Gigabyte 6800xt master type-c

主机配置：
CPU：i9 9900K

主板：ROG Maximus XI Hero WiFi z390

内存：金士顿 predator DDR4 4000MHz 8Gx4

显卡：技嘉 Auros 6800XT Master Type-C

SSD：三星 SM961 256G

散热：AlphaCool 240 一体水冷

显示器：LG 27UD88 4K

以太网：Intel i210T1

无线蓝牙：BCM 943602CDP

声卡：板载 ALC1220 + 独立 AudioEngine D1

压缩包已删除白果三码，随机生成了。机型smbios选取的 iMac 19,1

usb端口定制：前面板3.0借口舍弃了2.0功能，后面板usb-typc舍弃，边上的usb3.1舍弃，不然端口数量不够用。

基本完美了。
